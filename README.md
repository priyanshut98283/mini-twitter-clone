# Mini-Twitter-clone

## Features

- Sign up/ Sign in via JWT Token
- Customize account
- Tweet
- Comment
- Retweet
- Follow user
- Like tweets and comments
- Edit tweets and comments
- Delete tweets and comments

### Prerequisites

- node
- npm
- mongodb

# Install server dependencies

```
$ cd server
$ npm install
```

# Install client dependencies

```
$ cd client
$ npm install
```

### Run the app

1. Start the server

```
$ cd server
$ nodemon server.js
```

2. Start the client

```
$ cd client
$ npm start
```
